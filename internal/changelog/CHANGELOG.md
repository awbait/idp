# Журнал изменений

Заметные изменения портала. Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект придерживается семантического версионирования.

## [Unreleased]

### Добавлено
- Страница "О портале": версия сборки, полезные ссылки и журнал изменений по версиям.
- Проверка уникальности ресурсов инстанса в namespace (resource identity): два заказа одного чарта больше не могут отрендерить пересекающиеся имена ресурсов в один namespace и подраться за них в Argo CD.

### Исправлено
- Имя Argo CD Application теперь включает чарт (`team-chart-service`): два разных чарта с одинаковым `service_name` в одном namespace больше не конфликтуют за один объект Application.
