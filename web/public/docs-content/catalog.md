# Каталог сервисов

Каталог - это список сервисов, доступных для заказа. Он живой: новые версии сервисов появляются автоматически, как только их публикует платформа.

## Категории

Сервисы сгруппированы по категориям (например, сетевые шлюзы). В левом меню видны только те категории, в которых есть хотя бы один опубликованный сервис с формой заказа.

## Карточка сервиса

В карточке сервиса вы найдёте:

- описание и назначение сервиса;
- список доступных **версий**;
- кнопку **Заказать**.

## Версии

Каждый сервис выпускается в нескольких версиях. По умолчанию предлагается последняя стабильная. Если нужна конкретная версия - выберите её перед заказом. Уже заказанный сервис можно позже **обновить** до новой версии через действие на странице заказа.

| Что | Где |
|-----|-----|
| Доступные сервисы | Каталог -> категория |
| Версии сервиса | Карточка сервиса |
| Заказать | Кнопка «Заказать» в карточке |
