# Namespace

Изолированное пространство имён в кластере под сервисы команды. Это базовая единица платформы: внутри namespace разворачиваются остальные управляемые сервисы. При заказе настраиваются доступы команды (RBAC), а при включённом Service Mesh - метки Istio и сетевые политики по умолчанию.

> [!NOTE]
> Сервис планируется к добавлению. Здесь появятся параметры заказа и сценарии использования.
