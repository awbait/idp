# Быстрый старт

Этот гайд проведёт от входа в портал до первого работающего сервиса.

## Шаг 1. Вход

Откройте портал и войдите через Keycloak своей учётной записью. После входа в верхней панели проверьте выбранный **проект** (группа `team-*`) - заказ оформляется от его имени.

## Шаг 2. Выбор сервиса

Откройте [Каталог сервисов](catalog) в левом меню. Каталог сгруппирован по категориям. Выберите нужный сервис и откройте его карточку, чтобы увидеть описание и доступные версии.

## Шаг 3. Заказ

Нажмите **Заказать** и заполните форму. Поля и подсказки формируются автоматически из схемы сервиса, обязательные поля помечены. Подробнее - в разделе [Заказ сервиса](ordering).

## Шаг 4. Отправка

После отправки заказ попадает в **Список заказов**, а портал оформляет его как изменение в Git (merge request). Сервис развернётся после того, как этот MR будет влит. Что значит каждый статус, описано в разделе [Статусы и развёртывание](statuses).

## Шаг 5. Развёртывание

Отправленный заказ оформляется как изменение в Git (merge request), и после его слияния платформа разворачивает сервис в кластере. На странице заказа виден прогресс и итоговое состояние.

> [!NOTE]
> Первый заказ удобно делать на тестовом сервисе - так вы пройдёте весь путь без риска для рабочих окружений.

Готово. Дальше стоит изучить [Каталог сервисов](catalog).
