# Virtual Machines

Заказ виртуальных машин для нагрузок, которые не упакованы в контейнеры: legacy-приложения, специфичные ОС, изолированные окружения. ВМ управляются рядом с остальными сервисами платформы.

> [!NOTE]
> Сервис планируется к добавлению. Здесь появятся параметры заказа и сценарии использования.
