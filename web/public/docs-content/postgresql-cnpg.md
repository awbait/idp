# PostgreSQL (CNPG)

Управляемый PostgreSQL на операторе CloudNativePG: альтернативная реализация отказоустойчивого кластера PostgreSQL с cloud-native подходом к резервному копированию и восстановлению.

> [!NOTE]
> Сервис планируется к добавлению. Здесь появятся параметры заказа и сценарии использования.
