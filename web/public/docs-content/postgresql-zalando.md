# PostgreSQL (Zalando)

Управляемый кластер PostgreSQL на операторе Zalando (Patroni/Spilo): реляционная СУБД с репликацией и автоматическим переключением при отказе (HA). Подходит для основной транзакционной нагрузки приложений.

> [!NOTE]
> Сервис планируется к добавлению. Здесь появятся параметры заказа и сценарии использования.
